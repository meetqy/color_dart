# color_dart

display color in development

## 效果图
![](./color_dart.png);

## 使用

* hex(String hexColor)

```
hex('#222);
hex('#ff0000');
```

* rgba(int r, int g, int b, double a)

```
rgba(0,0,0,1);
```
